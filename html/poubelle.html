PRÉSENTATION DU PROJET

Nous avons réalisé ce projet dans le cadre du cours de PPE (Programmation et Projet Encadré), pendant le premier semestre de notre M1 en Master TAL.

Le but de ce projet est de répertorier, puis d'analyser la "vie multilingue" du mot "ouverture" sur le web. Pour réaliser ce projet, nous avons travaillé avec 4 langues: le français, le chinois, le coréen et le catalan. Nous avons commencé par trouver chacun 50 urls dans lequel notre mot apparaissait au moins une fois, le but final étant d'analyser ses contextes d'apparition. Une fois les urls récoltés, nous avons chacun créé un tableau (grâce à un script bash), dans lequel figurent les informations suivantes: une colonne pour les urls et, pour chaque url, le code http qu'elle renvoie (idéalement 200), l'encodage de la page, son nombre total de mots, le nombre d'occurrences du mot visé (ici "ouverture") dans la page, les contextes du mot (la/les ligne.s qui précède.nt et qui suit.vent), et un lien vers un concordancier.
